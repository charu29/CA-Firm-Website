Calculates Income Tax Liability for Salaried Persons of India.

== Description ==

This plugin calculates Income Tax Liability for Salaried Persons of India (2nd most populated country in the world).

This calculator is built with latest Income Tax Laws/Rules of India. It is at present capable to calculate income tax liability from assessment year 2010-11 onwards. Older years may be implemented upon request of users.

For a demo visit: http://wbxpress.com/income-tax-calculator/

== Installation ==

1. Delete previous installation, if you are running this plugin already.
2. Upload `income-tax-calculator-india` folder to `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Create a New Post/Page and name it "Income Tax Calculator".
5. Select "Text" view and type "[\*it_cal\*]" without quote.
6. Publish the Post/Page.
7. View the Post/Page in your Browser. You are done.

== Frequently asked questions ==

= Where is the settings menu of Income Tax Calculator Plugin?  =

There is no settings needed to use this calculator. Everything is inbuilt and conforming to the Income Tax Rules of Govt. of India.

= How to use this Plugin?  =

To use this plugin you have to insert a short code [\*it_cal\*] in your post/page.


= How can I be sure the calculation is correct?  =

For cross verification you may calculate Income Tax Calculate Liability manually as per Income Tax Rules provided in the demo page: http://wbxpress.com/income-tax-calculator/


= How to uninstall this plugin?  =

Deactivate the plugin through 'Plugins' menu and Delete. This plugin do not use your Wordpress database. So dont worry simple delete this plugin for a complete removal.
