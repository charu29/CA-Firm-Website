== Description ==
**Beamer** is a smart and easy-to-use newsfeed and changelog you can install on your site or app to announce relevant news, latest features, and updates.

This little announcement helper will get you up to 10x more user engagement and will help users learn new features and changes 3x faster. Use images, videos and gifs to make your announcements visual and engaging.

Beamer helps you collect user feedback on your announcements so you can make better decisions based on real data. And best of all it can be installed easily with this WordPress plugin - no code required.

== Installation ==
1. Go to your [Beamer dashboard](https://app.getbeamer.com/home) and copy the **product ID** from the top bar. If you don't have a **Beamer account** create one for free [here](https://app.getbeamer.com/signup).
2. Go to your WordPress admin panel and install the Beamer plugin.
3. Go to **Settings > Beamer Settings** and paste your product ID in the proper field. Save your changes.
4. Enjoy

Beamer offers many customization options. For more information on how to install and customize your Beamer feed for WordPress go to [this tutorial](https://www.getbeamer.com/blog/how-to-use-beamer-as-a-wordpress-plugin/). Learn more about beamer in our [Developer Documentation Page](https://www.getbeamer.com/docs/).
